# Dani Morais — Landing Page

Landing page oficial da cantora Dani Morais, construída em [Next.js](https://nextjs.org) (App Router) com TypeScript.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Stack

- Next.js 16 (App Router, Turbopack)
- React 19 + TypeScript
- CSS global customizado (sem framework de UI)
- `next/image` para otimização das fotos
- `next/font` (Fredoka + Outfit via Google Fonts)

## Build de produção

```bash
npm run build
npm start
```
